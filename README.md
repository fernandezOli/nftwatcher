# NFT Watcher

This project is monitoring of NFT transfers when using NFT as an access pass to a service

## Motivation

We want users to interact with a service using NFT like pass.

## Clone this repo

Clone this Github repo, cd into it and:

```shell
npm i
```

## Run

```shell
npm start
```

## Test contract

```shell
npx hardhat test
```
